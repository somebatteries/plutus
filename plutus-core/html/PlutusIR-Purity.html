isPure :: ToBuiltinMeaning uni fun => BuiltinVersion fun -> (name -> Strictness) -> Term tyname name uni fun a -> Bool Source #

Will evaluating this term have side effects (looping or error)? This is slightly wider than the definition of a value, as it includes applications that are known to be pure, as well as things that can't be returned from the machine (as they'd be ill-scoped).

firstEffectfulTerm :: Term tyname name uni fun a -> Maybe (Term tyname name uni fun a) Source #

Try to identify the first sub term which will be evaluated in the given term and which could have an effect. Nothing indicates that we don't know, this function is conservative.

View a Term as a BuiltinApp if possible.

isSaturated :: forall tyname name uni fun a. ToBuiltinMeaning uni fun => BuiltinVersion fun -> BuiltinApp tyname name uni fun a -> Maybe Bool Source #

Is the given BuiltinApp saturated? Returns Nothing if something is badly wrong and we can't tell.
